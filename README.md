# BIOPOD

BIOPOD is an AI-powered IoT-based bio-conversion monitoring system.

## Hardware Used
- ESP32
- DHT22 (Temperature & Humidity)
- MQ135 (Air Quality)
- LCD Display
- Buzzer

## Features
- Real-time sensor monitoring
- MQTT-based data transfer
- AI-based health score
- Growth stage prediction
- Feed vs fertilizer tracking
- Location weather & AQI comparison
- Single-page dashboard
- AI Assistant (AIRA)

## Tech Stack
- MQTT
- Node.js + Express
- MongoDB
- React (Vite)
- Tailwind CSS
